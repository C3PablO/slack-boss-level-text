# Slack Boss Level Text

A web app that converts plain text into Slack letter glyph emoji codes. Type your message, pick a style, copy the output, and paste it into Slack for bold, eye-catching messages.

https://slack-boss-level-text.netlify.app/

## Features

- Converts letters (A-Z), numbers (0-9), and special characters (!?#@) into Slack emoji codes
- Two glyph styles: **white** and **yellow**
- Live preview of your text
- One-click copy to clipboard

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)

### Install and Run

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
npm run preview
```

## Deployment

The project includes a `netlify.toml` config and deploys to [Netlify](https://www.netlify.com/) out of the box.

## Tech Stack

- React 18
- TypeScript
- Vite
